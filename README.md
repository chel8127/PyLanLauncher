# PyLanLauncher
🇺🇸: Welcome to my launcher for Minecraft! This launcher is completely free. In it, you can play without a license (offline) and with a license (Microsoft). You can download mods there, choose a loader, create installations, and much more!  The launcher is inspired by PrismLauncher
